About service

About functionality:

  1. You can download and see all vacancies and summaries
  2. use filters and search input for more flexible search
  3. Can register or login for addition functions
  4. After registration you have your own personal cabinet, where appears new functions
  5. After registration you can add, edit or remove you own application in vacancy or summary
  6. And of course there is dark mode and language switcher for 2 languages
  7. I'm using firebase like backend
  8. Maybe I will implement something new...
